# 创建 next.js 项目

## 直接在命令行创建

```shell
yarn create next-app rubik-cube --typescript
cd rubik-cube
yarn dev
```

在浏览器打开 `http://localhost:3000/`，看到下面的图就说明项目创建成功了。
