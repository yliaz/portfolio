## 安装

在[官网](https://code.visualstudio.com/)下载安装适合自己的版本。

## 将 `code` 命令配置到终端

1. `⌘` + `shift` + `P`，显示所有命令
2. 输入 `path`，找到 `在 path 中安装 code 命令`
3. 回车（`↩`）

## 基本快捷键

1. `Ctrl` + `B`，打开/关闭侧边栏
2. `⌘` + `Shift` + `P`，查找所有命令
3. `⌘` + `P`，查找
4. `Ctrl` + ```，打开终端

> **查看所有快捷键的方法：**
> 1. `⌘` + `Shift` + `P`
> 2. 输入 `keyboard`
> 3. 找到 `Open Keyboard Shortcuts`
