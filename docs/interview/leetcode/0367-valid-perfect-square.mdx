# 367. 有效的完全平方数

## 题目

给定一个 正整数 `num` ，编写一个函数，如果 `num` 是一个完全平方数，则返回 `true` ，否则返回 `false` 。

## 代码

<Tabs groupId="solutions">

```ts
function isPerfectSquare(num: number): boolean {
    let left = 0
    let right = num
    while(left <= right) {
        const mid = left + Math.floor((right - left) / 2)
        const temp = mid * mid
        if (temp === num) {
            return true
        } else if (temp < num) {
            left = mid + 1
        } else {
            right = mid - 1
        }
    } 
    return false
};
```
